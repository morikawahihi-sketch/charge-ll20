## GitHub Desktopでの更新方法
1. このフォルダ内のファイルをすべてコピー
2. GitHub Desktopの「Show in Explorer」で開いた charge-ll20 フォルダへ貼り付け
3. 同じ名前のファイルは「置き換える」
4. GitHub Desktopで Summary に更新内容を入力
5. Commit to main
6. Push origin
